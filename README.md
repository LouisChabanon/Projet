# Simulation potager

Ce programme simule un potager contenant des parcelles, des plantes, des insectes et des dispositifs

### Installation
Python 3 est requis

```bash
git clone https://github.com/louischabanon/Projet
cd Projet
pip install -r requirements.txt
```
